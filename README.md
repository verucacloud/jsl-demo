# 🛒 Aplikasi Laporan Keuangan - PHP Native MVC

Aplikasi manajemen inventaris dan keuangan sederhana yang dibangun menggunakan arsitektur **MVC (Model-View-Controller)** tanpa framework (PHP Native). Proyek ini dibuat untuk mendemonstrasikan pemahaman alur kerja OOP dan struktur database MySQL/PostgreSQL.

## 🚀 Fitur Utama
- **Autentikasi Aman:** Login sistem dengan session.
- **Manajemen Barang:** CRUD (Create, Read, Update, Delete) data barang toko.
- **Laporan Keuangan:** Pencatatan pemasukan otomatis.
- **Arsitektur MVC:** Pemisahan logic, data, dan tampilan yang rapi.
- **URL Ramah (Pretty URL):** Menggunakan `.htaccess`.
